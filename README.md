# BuildIso Encrypt
With BuildIso Encrypt you can encrypt your files. Website: https://buildiso.com

# About BuildIso Encrypt
BuildIso Encrypt is encryption and decryption software.

# How to decrypt?
To decrypt, make sure that the software used to encrypt was indeed BuildIso Encrypt. Then launch BuildIso Encrypt (by double-clicking on it), click on Browse and navigate to the file with the ENC extension, then enter your password (created when encrypting the file) or enter one of the 10 backup keys (created when encrypting the file). After that, wait for it to decrypt, then a popup message will inform you.

# How to encrypt?
To encrypt, launch BuildIso Encrypt (by double-clicking on it), then click on Browse and navigate to the file with the extension (txt, iso...). Next, type your password (whatever you want). After that, a popup message will appear asking if you want to save the 10 recovery keys (we recommend saving them).

# What should I do if I forget my password?
If you have lost your password, contacting support will be useless and we cannot help you (because the software is offline and we cannot see your passwords, fortunately). If you have lost it, try to find the file BuildIsoEncrypt_RecoveryKeys.txt (if you saved the file at the time of encryption). If you don't have it, it is very difficult, if not impossible, to recover the file because the software uses 300,000 iterations of PBKDF2, salt, tools from using System.Security.Cryptography, and secure deletion. So the only way is either to recover the password or the UUIDs.
